Button and Alert Box

Buttons can be placed to allow the user to interact with the website. They can cause functions to begin to run such as adding an item to a To-Do List or in the case of the button to the left allow a push alert to appear.

Push Alerts provide the user with information. They can appear upon entering a webpage or when a set action occurs such as pushing the button to the left. If validation is required, such as in forms push alerts are useful in alerting the user of any issue that may have occured.
